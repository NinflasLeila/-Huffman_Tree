#  Projet Huffman Tree

## Contexte
Ce projet consiste à implémenter l’algorithme de compression de **Huffman** en Python.  
Il permet de comprendre les étapes clés de la compression sans perte : analyse des fréquences, construction de l’arbre, encodage et décodage.

## Exécution
Une application interactive permet de visualiser l’implémentation de l’arbre de Huffman et le processus de compression.

streamlit run app.py
